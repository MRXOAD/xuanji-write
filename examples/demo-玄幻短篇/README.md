# demo-玄幻短篇

修仙题材最简模板,3 章示例 + 完整设定集骨架。**3000 字以内能让你跑出第一章**。

## 这本"书"的设定

- 题材:修仙(也可改西幻 / 高武)
- 主角:沈砚之,寒门剑修,父母被门派灭口
- 反派:卓青云,门派副掌门,主角生父血亲兄弟
- 主线:沈砚之入门 → 拜师 → 追查父母死因 → 翻盘灭门派
- 长度:全书 50 章,demo 提供前 3 章

## 用法

```bash
cp -r examples/demo-玄幻短篇 ~/.../books/我的修仙
cd ~/.../books/我的修仙

# 改 设定集/ 三个文件:角色约束、角色语料白名单、世界观
vim 设定集/角色约束.md

# 第 4 章大纲已留模板,改完跑
python webnovel-writer/scripts/webnovel.py llm draft --chapter 4
```

## 文件清单

| 路径 | 用途 |
|---|---|
| `设定集/角色约束.md` | 角色锚点,自动注入 system prompt |
| `设定集/角色语料白名单.md` | 主要角色名单,extract_character_voice 用 |
| `设定集/世界观.md` | 修仙体系、门派、地理 |
| `大纲/总纲.md` | 全书主线 |
| `大纲/阶段支架.md` | 50 章分段 |
| `大纲/第1章-XXX.md` | 第 1 章细纲 |
| `正文/第0001章-XXX.md` | 示例章 |
| `.webnovel/state.json` | 已初始化的进度文件 |
