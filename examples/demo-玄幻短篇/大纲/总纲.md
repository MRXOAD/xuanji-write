# 总纲

## 一句话

寒门剑修沈砚之入青鸾宗,边修炼边查父母死因,最后掀翻自己叔父执掌的门派一角。

## 主线

| 卷 | 章 | 主线 | 主爽点 |
|---|---|---|---|
| 1 | 1-15 | 入门考核 + 拜师 | 剑骨惊门派 |
| 2 | 16-30 | 第一次秘境 + 发现父母旧线索 | 秘境捡漏 |
| 3 | 31-50 | 当面对峙叔父 + 阶段性翻盘 | 庭审改谱 |

## 角色弧线

- 沈砚之:孤身入门 → 师徒情 → 同门情 → 仇情交织 → 选择不复仇而是改谱
- 卓青云:暗中提防 → 试探 → 明面交锋 → 退而不杀
- 林霜白:监视者 → 矛盾 → 倒戈帮主角

## 题材锚定

传统修仙,练气/筑基/金丹/元婴/化神。无系统、无穿越、无现代元素。
