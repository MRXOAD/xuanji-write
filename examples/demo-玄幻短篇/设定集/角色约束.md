# 角色约束

> 自动注入 system prompt。每行一条,编号续上通用规则。
> 改这本书前先把这里的角色名换成你自己的。

- 主角固定为沈砚之,寒门剑修,父母被门派灭口,16 岁拜入青鸾宗;不要写成天才/转世/穿越者。
- 沈砚之的设定是"剑骨锋利但灵根杂",修炼难度高于同辈;不要给他"逆天灵根""上古血脉"。
- 反派卓青云固定是青鸾宗副掌门,主角生父的同胞兄弟,当年灭门主谋;不要把他写成纯小人,他自有其为门派的算计。
- 已死亡的人物(沈父沈母)不能在当前场景直接现身说话,只能通过回忆/旧物登场。
- 师父一脉:萧执玄(主角直系师父,半隐退),梅清远(师叔,管事),不要互换两人定位。
- 师姐林霜白动机复杂,既被卓青云授意监视主角,又对主角动了真情;不要写成纯反派或纯爱慕者。
- 题材是传统修仙,不要写"系统签到""穿越"这类元素。
- 主角第一卷只在青鸾宗内活动,不要让他跨州/远行。
- 修炼境界用"练气/筑基/金丹/元婴/化神"五阶,不自创新境界。
- 灵根设定固定:剑骨杂灵,不属传统五行单灵根;主角的剑骨是天赐,但反过来限制其它灵根发展。
- 章末钩子用"信息钩 + 谜题钩"为主,少用情绪钩。
