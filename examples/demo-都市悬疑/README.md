# demo-都市悬疑

跟修仙差异最大的 demo。现代背景,网约车司机 + 连环失踪案。

## 设定

- 主角:陈晓白,32 岁,前 S 市刑警五年前因一桩失败案件离职,现网约车司机。
- 女主:林栀,深夜电台 DJ,陈晓白离职那年的报案人之一(姐姐失踪)。
- 反派:周鸣,社区医生,陈晓白同住一栋楼的邻居,真正的连环犯。
- 主线:陈晓白接到一单乘客中途失踪 → 旧案重启 → 凶手就在身边。
- 长度:50 章三卷,demo 提供前 3 章。

## 题材锚点

- 现代都市,2020 年代,真实城市感(地铁/高架/24 小时便利店/外卖平台)
- 写实派悬疑,不写超能力,不带玄学
- 警察 / 法医 / 司法流程要尊重程序

## 用法

```bash
cp -r examples/demo-都市悬疑 ~/Projects/.../books/我的悬疑
cd ~/Projects/.../books/我的悬疑
# 改 设定集/角色约束.md 换成自己的人物
python webnovel-writer/scripts/webnovel.py llm draft --chapter 1
```
