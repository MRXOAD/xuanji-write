# 世界观

## 城市

- S 市,长江下游某地级市,常住人口 800 万。
- 主角活动区域:老城区(20 年以上居民楼) + 高新区(新建写字楼/商场) + 滨江(夜生活)
- 关键地标:槐安里(主角和反派同住的老小区)、南桥(深夜空旷桥)、人民医院、24 小时便利店"夜半"

## 时代

2025-2026 年。手机/微信/外卖/网约车日常化。

## 警务设定

- S 市公安局刑侦支队,主角原所属
- 失踪 24 小时立案,72 小时后专案组介入
- 陈晓白离职后无警察身份,只能通过前同事李建国侧面参与
- 法医徐青在市公安局法医中心

## 案件方法论(给 LLM 参考)

写案件细节时遵守:
1. 失踪案不直接立"杀人案",至少 3 个间接证据 + 1 个直接证据才能定性
2. DNA 比对需 3-7 工作日
3. 监控调取需出警察证 + 物业配合
4. 网约车订单数据警方可调,但需要书面手续

## 不写的事

- 神秘组织 / 跨国阴谋 / 政治背景
- 主角持枪
- 反派一次绑架多人
- 主角和警方对抗
