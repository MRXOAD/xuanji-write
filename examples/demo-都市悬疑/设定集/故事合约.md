# 故事合约(Story System)

> 这份文件被 webnovel-codex 自动注入 system prompt,定义全书宪法,优先级高于单章大纲。
> 改这份文件相当于改全书定调,不到必要不动。

## 核心冲突

陈晓白(前刑警 / 网约车司机)在五年前办砸的林雪失踪案与本周 S 市新发的连环案中找到同一种凶手手法。他要在体制外、不靠警权,把这个跨度 5 年的链路重新挖出来。

## 题材合约(给读者的体验承诺)

- **悬疑+程序**:每两章一条新线索,每六章一次小翻案
- **都市冷感**:城市夜晚 / 网约车 / 老旧居民楼 / 凌晨电台
- **不堆血腥**:恐怖来自"差一点就抓到"的张力,不来自尸体描写
- **不写超自然**:所有怪事最后必有现实解释

## 主角弧

- 起点(1-15 章):自我封闭,只想跑车糊口,逃避旧案
- 中段(16-30 章):被新案逼回旧案,以"顾问"非警察身份介入
- 终段(31-50 章):亲手抓到凶手,但接受自己不再回到体制内

## 主反派弧(周鸣)

- 表面:S 市社区医院年轻医生,温和,邻居印象好
- 暗面:五年前林雪案就是他做的,这五年用医院身份接近独居女性
- 触发:林栀的"姐妹寻人"电台节目让他重新活跃,他主动给电台投稿"猎物自白"

## 节奏曲线

- 开篇钩(第 1 章):末班车女乘客失踪 + 口红刻字"L. 雪"
- 第一拐点(第 8 章):陈晓白发现耳环数字 805 = 林雪当年丢的耳坠
- 中点反转(第 25 章):陈晓白怀疑过的同事其实是无辜
- 最低点(第 40 章):凶手拿到林栀,陈晓白警权被吊销
- 高潮(第 48 章):12 楼对峙,无武器
- 收束(第 50 章):凶手被捕,陈晓白离开警局,继续跑车,给林栀寄一封无名信

## 主线 / 副线

- 主线:连环失踪案
- 副线 A(感情):林栀和陈晓白通过电台 / 邮件 / 信件慢慢靠近,但全书不超过一次见面
- 副线 B(职场):李建国(刑警队老搭档)在体制内策应陈晓白
- 副线 C(过去):林雪失踪案的真相通过本周新案逐步浮现

## 节奏强约束

- 每 3 章必有一次"案情推进或回退",不允许 5 章以上原地打转
- 凶手视角章节最多占全书 1/5,不允许大量篇幅
- 林栀视角的电台旁白每 4-6 章用一次,作为节奏调节

## 写作纪律

- **不写"丹田 / 经脉 / 修为"** — 题材是现代悬疑,不是修仙
- **主角名永远是"陈晓白"**,不要写成"陈晓白"以外的同姓变体
- **林雪是"林栀"的姐姐**,这两个名字不要互串
- **周鸣的医生身份直到第 18 章才能被读者怀疑**,前 17 章他只是邻居 NPC
