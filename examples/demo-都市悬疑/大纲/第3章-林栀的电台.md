# 第3章:林栀的电台

- 场景:当晚 23:00 后,陈晓白的车里,FM 91.6《深夜不打烊》节目。
- 核心物件:电台一档"姐妹寻人"特别栏目(每周一次),今晚的来信。
- 本章冲突:节目里林栀念了一封听众来信,内容是"姐姐五年前在 S 市南桥附近最后一次出现",陈晓白在车里愣住——这是他正在追的失踪案延伸到的旧案。
- 可见赢面:陈晓白第二天联系电台找林栀,但林栀拒绝面谈,只愿意通过电台或邮件;陈晓白通过李建国侧面查到林栀就是 5 年前林雪案的报案家属(妹妹)。
- 必碰线索:林栀这五年坚持每周一次"姐妹寻人"栏目,从未停过;最近两期来信里,有一封提到"南桥""口红""周二凌晨"——和昨晚陈晓白的失踪订单时间地点完全吻合;来信落款只写了一个字母 "Z"。
- 人物关系:林栀首次出场(电台声音 + 拒接面谈),陈晓白没透露自己是前刑警;李建国不愿大张旗鼓查电台听众。
- 章末钩子:陈晓白回家走楼道时,12 楼周鸣家门半掩,他听见里面收音机正放着同一档电台,周鸣的声音对着收音机自言自语:"她这周又收到信了。"
- 主导Strand:quest。
